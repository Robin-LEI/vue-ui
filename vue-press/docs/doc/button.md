# 属性 Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 说明 |
| ------ | ------ | ------ | ------ | ------ | ------ |
| @click | 点击事件 | event | - | - | - |
| size | 按钮尺寸 | string | medium / small | medium | 大多数情况下使用默认尺寸即height为34px，某些情况下需要使用height为30px，此时设置属性size="small"即可|
